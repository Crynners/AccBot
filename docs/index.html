Security First

AccBot is built on a decentralized, security-first architecture. Your data stays on your device.

AES-256-GCM Encryption API credentials encrypted with hardware-backed Android Keystore
Local Storage Only No data is ever uploaded to any server
Direct API Calls Communication goes directly to exchanges via HTTPS
Biometric Lock Optional fingerprint or face recognition to open the app
Open Source Fully auditable codebase under MIT license
No Cloud Sync No analytics, no tracking, no ads, no telemetry

How It Works

1

Connect

Add your exchange API keys with trading-only permissions. Scan QR codes or enter manually.

2

Configure

Set your crypto, amount, frequency, and strategy. Choose from Classic, ATH-based, or Fear & Greed DCA.

3

Accumulate

AccBot runs in the background and executes purchases on your schedule. Track performance in the portfolio view.
